Support

JournalBear is built and supported by one person. The fastest ways to get help:

  • Email yrahul3910@gmail.com
  • Open an issue on GitHub

I forgot my password

JournalBear cannot recover it — that's a promise, not a limitation. Your password never leaves your device and is the only key to your journal. Without it, the encrypted file cannot be opened by anyone, including us. Keep your password somewhere safe, such as a password manager.

Moving my journal between devices

Your whole journal is one .zjournal file. Put it anywhere the Files app can reach (iCloud Drive, for example) and open it from JournalBear on another device — iPhone or desktop. The same password opens it everywhere.

Backups

Back up the .zjournal file like any other document — copy it to a drive, a synced folder, or your backup tool of choice. The file is fully encrypted, so it is safe at rest wherever you keep it.

Something looks wrong

If an entry renders oddly or the app misbehaves, an issue on GitHub with the steps you took is the most useful thing you can send. Please never include your journal file or password in a report.
